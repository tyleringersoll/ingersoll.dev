# ingersoll.dev

Personal portfolio website built with Vue.js 3, hosted on Netlify.

## Tech Stack

- **Vue.js 3** - Composition API with `<script setup>`
- **Vue Router** - Client-side routing
- **Pinia** - State management
- **Sass** - Styling
- **Jest** - Unit testing
- **Netlify** - Static hosting

## Project Setup

```bash
npm install
```

## Development

```bash
npm run serve
```

## Production Build

```bash
npm run build
```

## Testing

```bash
npm run test:unit
```

## Linting

```bash
npm run lint
```
